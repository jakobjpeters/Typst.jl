Typstry.@typst_str — Macro
@typst_str(s)
typst"s"

Construct a string with custom interpolation and without unescaping. Backslashes (\) and quotation marks (") must still be escaped.

The syntax for interpolation is a backslash \, followed by the Mode to typstify the interpolated value, and finally a Julia expression enclosed in parentheses (x). The syntax typst"\(x)" is equivalent to typst"\markup(x)".

Warning

See also the performance tip to avoid string interpolation for I/O.

Examples

julia> x = 1;

julia> typst"$1 / x$"
typst"\$1 / x\$"

julia> typst"\(x) \(x + 1)"
typst"1 2"

julia> typst"\\(x)"
typst"\\\\(x)"

julia> typst"\ \\"
typst"\\ \\"
