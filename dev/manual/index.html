Typstry.@typst_str — Macro
@typst_str(s)

Construct a string with custom interpolation and without unescaping. Backslashes (\) and quotation marks (") must still be escaped.

Use $$ to interpolate values. The $ symbol is not used because Typst uses that symbol to start and end a math mode block.

Examples

julia> x = 1;

julia> typst"$1 / x$"
"\$1 / x\$"

julia> typst"$$x $$(x + 1)"
"1 2"

julia> typst"\$$x"
"\$\$x"

julia> typst"\\"
"\\"

Typstry.@typst_cmd — Macro
@typst_cmd(s)

Return a Cmd such that run(typst`$s`) is equivalent to shell> typst $s.

Note

This macro does not yet support interpolation.

Examples

julia> run(typst`help`);

Typstry.render — Function
render(::String; input = "input.typ", output = "output.pdf", open = true)

Render the given string to a document.

This function generates two files. The first is the input, which contains the Typst document. The second is the output, which is rendered from the input using Typst's compile function.

The document format is inferred by the file extension of output. The available formats are pdf, png, and svg.

If open, output will be opened using the default viewer.

Note

This is designed to generate a document with little effort. For more advanced useage, see typst.

Examples

julia> render(typst"$x ^ 2$");

julia> render([1, 2, 3, 4]);
source
render(elements...; delimeter = " \\n", kwargs...)

Examples
